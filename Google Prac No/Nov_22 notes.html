Functions

Functions are blocks of code designed to perform a particular task when "something" invokes it(calls it).
Function Syntax:
function (parameters){
     lines of code;
};

Functions can be named. This will allow you to invoke it(call it) later on in your code.
There are two ways to name a function

  • Placing the name between the function keyword and the parameters.
  • Assigning the function to a variale.
function functionName(parameters){
     lines of code;
}
var functionName = function(parameters){
     lines of code;
}

Parameters and Arguments

Parameters are placeholders for arguments. They act as local variables for the function. Not all funtions will require parameters. Arguments are the values given when a fucntion is called.

Return Statement

Functions can return a value by using a return statement. To do so, the return keyword must be at the end of your code followed by the value you want to return.

Scopes in Function

Variables created inside a fucntion can not be used outside of the function. The same foes for the functions' parameters.

Objects

Objects, like arrays, are variables that can hold multiple values. But instead of the values being identified with indices, they are identified with names. These are called properties.

  • Objects are created with curly braces { }, values go within them
  • Objects names and values are separated with colons:
  • Add multiple properties by separating them with commas ,
Syntax:

var obj = {
     name1: value1, name2: value2 };

Practice: Creating a person object

function makePerson(){
     var person = {
       name: 'Trey',
       age: 72,
       grade: [50, 84, 87, 90]
     }
     return person;
}

var p = makePerson();
p.name = 'White';

You can add a new property to an object by:
p.favoriteSubject = "Drawing";

console.log('My name is ' + p.name + ', I am '+ p.age+ ' years old. My first grade is ' + p.grade[0] + ".");

function makePerson(name1, age1, grades1){

var p = makePerson('Thomas', 200, [60, 90, 99]);

Data types used in objects

Objects can hold all data types. From primitive types, like strings and numbers, to complex types, like arrays, functions, and even another object.

     var baby = {
     name: "John Cena",
     age: 2,
     canWalk: false,
     favoriteFood: ['Milk', 'Squash', 'Peas'],
     cry: function(){console.log("wahh");}
     }

Properties that uses functions as its value are referred to as methods, actions that can be performed on objects.

Why Use Objects

Consider the baby example. All babies have the same properties, but the property values differ from baby to baby. All babies have the same methods, but the methods are performed at different times.

Accessing Objects Properties and Methods

There are two ways of accessing properties/methods of an object.

var dog = {
     breed: "pitbull",
     speak: function(){console.log("BARK");}
}

  • Dot notation (Most widely used)
            dog.breed //returns "pitbull"
            dog.speak() //prints BARK
  • Square brackets
            dog.[breed] //returns "pitbull"
            dog.["speak"]() //prints BARK

For/in Loop and Objects

For/in loop work very well with object

var pets = {dog: "pug", cat: "ragdoll", cow: "angus"};

for(var i in pets){
//i will return the properties of the object ie. dog, cat, cow
        console.log(i)

for(var i in pets){
